Реестр товаров и услуг для домашних животных.

Этот сайт является продолжением курсовой работы.

Целью являлась замена фронтенда на Windows Forms на одностраничное веб-приложение на Angular.

Стек технологий:

  • Angular - frontend
  • nginx - обратный прокси
  • ASP.NET Web Api - backend
  • MS SQL Server - база данных
  • Docker
